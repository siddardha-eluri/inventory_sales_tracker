# 📦 Inventory & Sales Tracker

A modern inventory and sales management web application built with **Next.js**, **Firebase**, and **Tailwind CSS**. The application helps businesses efficiently manage products, monitor stock levels, track sales transactions, and gain insights through a clean and responsive dashboard.

## ✨ Features

- 📋 Product Management (Add, Edit, Delete Products)
- 📦 Real-Time Inventory Tracking
- 💰 Sales Recording and Management
- 📊 Dashboard with Key Business Insights
- 🔍 Search and Filter Functionality
- 🔄 Firebase Integration for Real-Time Data
- 📱 Responsive UI with Tailwind CSS
- ⚡ Fast and Scalable Next.js Architecture

## 🛠️ Tech Stack

- **Frontend:** Next.js (App Router)
- **Backend & Database:** Firebase Firestore
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** Vercel

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/inventory-sales-tracker.git
cd inventory-sales-tracker
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open http://localhost:3000 in your browser to view the application.
